OpenFin Cloud Interop

OpenFin allows OpenFin applications to share Interop & FDC3 Context across the desktop using our Interop Broker pattern and OpenFin's desktop message bus. OpenFin's Cloud Interop solution ( @openfin/cloud-interop ) provides an interop broker override that can be added to OpenFin desktop platforms as well as OpenFin web platforms ( @openfin/core-web ). Once the cloud interop override is initialized with settings provided by OpenFin it will connect the platforms together through our cloud offering so that they can share contextual information. Please contact OpenFin if you would wish to do a PoC to evaluate our cloud offering so that you can get the required settings.
